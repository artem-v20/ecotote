# EcoTote 🌱

A one‑page responsive website showcasing eco‑friendly tote bags. This project
was developed as a team assignment during IT courses.

## 🚀 MVP Goals

- Responsive layout (mobile ≥320px, tablet ≥768px, desktop ≥1440px)
- Valid HTML5 & CSS3 (checked via [W3C Validators](https://validator.w3.org/))
- Semantic markup following HTML5 standards
- Optimized images (retina support, SVG sprite icons, favicon)
- Interactive elements styled with hover/focus effects

## 📂 Project Structure

- **Header** with logo, navigation, and contacts
- **Hero** section with main slogan and CTA
- **About** section describing eco‑bags
- **Advantages** section with benefits list
- **Assortment** product cards with prices and buy buttons
- **Gallery** of lifestyle images
- **Feedbacks** customer reviews ⭐
- **Support** contact form with validation
- **Footer** with navigation, social links, and copyright

## 🛠 Tech Stack

- HTML5, CSS3
- Google Fonts (Raleway, Roboto)
- SVG sprites
- Modern Normalize

## ▶️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ecotote.git
   ```
2. Navigate into the project folder:
   ```bash
   cd ecotote
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open the local server link in your browser (usually `http://localhost:3000`)
